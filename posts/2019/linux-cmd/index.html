tar

命令选项：

-z(gzip)      用 gzip 来压缩/解压缩文件
-j(bzip2)     用 bzip2 来压缩/解压缩文件
-v(verbose)   详细报告 tar 处理的文件信息
-c(create)    创建新的档案文件
-x(extract)   解压缩文件或目录
-f(file)      使用档案文件或设备，这个选项通常是必选的。

示例：

# 压缩
tar -zcvf abc.tar.gz abc
tar -jcvf abc.tar.gz2 abc

# 解压
tar -zxvf abc.tar.gz
tar -jxvf abc.tar.bz2
