Hugo 主题 Gemini

Posted by RoninRo on Mon, Jun 5, 2017

最近在https://www.linpx.com/上看到了博主的博客主题，感觉很漂亮，不过是Typecho 主题的。于是便用了周末的时间将它转成了Hugo的。 主题地址在：https://github.com/mjyi/Gemini

Pinghsu Theme

链接地址：https://github.com/chakhsu/pinghsu.

作者主页：https://www.linpx.com/.

响应式设计，对于移动设备也有良好的体验。

支持功能

  • 首页分页显示
  • 分类
  • 标签
  • 归档

使用主题

安装Hugo

  1. 安装Hugo,Mac用户可以使用Homebrew 安装：
1$ brew install hugo

其他平台下载地址：Hugo Releases

关于使用方法详见：

https://gohugo.io/

中文版

下载主题

使用Gemini主题，下载或者使用git clone主题，放到 Hugo站点的themes目录下。

1cd ~/yoursite
2git clone https://github.com/mjyi/Gemini.git themes/Gemini

配置文件

参考 exampleSite/config.toml,如下：

 1baseurl = "https://mjyi.github.io/Gemini/"
 2title = "Hugo Theme Gemini"
 3canonifyurls = true
 4paginate = 3 
 5languageCode = "en-us"
 6MetaDataFormat = "yaml"
 7theme = "Gemini"
 8googleAnalytics = "" # Google 统计
 9disqusShortname = "" #Disqus评论。
10
11[params]
12  Keywords = ""
13  Author = "Gemini"
14  cloudTieKey = "" #优先使用网易云跟帖
15  recentPostCount = 7 	#底部显示最新发表的文章数量
16  intro = ["I am a iOS Developer", "Work in Hangzhou"]  #Footer个人介绍。
17
18# 底部的社交信息
19[[params.social]]
20  title = "Email"
21  url = "mailto:itxuz@itxuz.com"
22[[params.social]]
23  title = "Github"
24  url = "https://github.com/mjyi"
25[[params.social]]
26  title = "RSS"
27  url = ""
28
29[[menu.main]]
30  name = "Archives"
31  url = "/post/"
32  weight = -200
33[[menu.main]]
34  name = "Tags"
35  url = "/tags/"
36  weight = -180
37[[menu.main]]
38  name = "About"
39  url = "/about"
40  weight = -160

关于单个文章的一些设置

 1+++
 2tags = ""
 3categories = ""
 4image = "/images/thumbs/0.jpg" #set a picture, or randomly select one in the folder images/thumbs
 5draft = true
 6comments = true
 7toc = true
 8author = ""
 9author_url = ""
10+++
  • tags ：文章标签。
  • categories ：文章分类
  • image：首页文章背景，不设置将从/images/thumbs 内的9张中随机选择一张。
  • comments：是否显示评论。
  • toc： 是否文章目录。
  • author,author_url：作者信息，显示于文章底部。不设置则使用Site.Params.Author
  • draft：是否是草稿，hugo默认不编译草稿，可使用 hugo server --buildDrafts调试。
