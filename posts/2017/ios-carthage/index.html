Carthage 的简单使用

Posted by RoninRo on Mon, Jun 19, 2017

Carthage 是和 Cocoapods 类似的第三方库管理工具。和 Cocoapods 相比，Carthage 仅导入 framework 文件，做到更少的入侵性。但是Carthage 仅支持动态框架，只支持iOS 8及以上版本或者任何的OS X系统。

使用Homebrew安装Carthage

1brew update
2brew install carthage

注意 如果已经安装过Carthage的二进制版本，则应删除 /Library/Frameworks/CarthageKit.framework

向项目中添加frameworks【iOS】

  1. 在项目工程根目录中新建Cartfile文件。

     touch Cartfile
  • carthage update命令获取并构建预编译的框架。
  • application targets -> General -> Link Frameworks and Libraries 从Carthage/Build文件夹中添加framework文件。
  • 在Build Phases选项，单击“+”图标，选择New Run Script Phase。创建一个运行脚本，您可以在其中指定shell（例如：/ bin / sh）
1/usr/local/bin/carthage copy-frameworks

将frameworks目录添加到 “Input Files”, 例如:

1$(SRCROOT)/Carthage/Build/iOS/Box.framework
2$(SRCROOT)/Carthage/Build/iOS/Result.framework
3$(SRCROOT)/Carthage/Build/iOS/ReactiveCocoa.framework
